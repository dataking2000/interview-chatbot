# AI Interview Coach Pro

An advanced AI-powered interview simulation platform with real-time feedback and analytics.

## Features
- AI-powered interview simulations
- Real-time performance analysis
- Speech recognition and analysis
- Comprehensive feedback and scoring
- Multiple career domains
- PDF report generation

## Deployment on Render

This app is configured for deployment on Render.com

## Local Development

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python app.py`
